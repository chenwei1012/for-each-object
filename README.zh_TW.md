# 一個Weixin Script (wxs) 對象迭代器

> 使用示例：
``` shell
var feo = require('forEachObj.wxs');

var obj = {
  // ...something object
}

feo.forEachObj(obj, function(key, value) {
  // ...something code
});
```
具體使用請詳見項目源碼。
